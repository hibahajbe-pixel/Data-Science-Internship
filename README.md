# personal-finance-manager

## Project summary
Command line personal finance manager for tracking and reporting expenses.

## Quick start
1. Open terminal at project root
2. Run `python -m src.main`

## Folder structure
```
personal-finance-manager/
├ src/
├ data/
├ reports/
├ docs/
├ tests/
├ screenshots/
├ README.md
└ requirements.txt
```

## Features
- Add expenses
- View all expenses
- Category wise summary
- Monthly reports
- Backup data

## Requirements
Python 3.7 or above
